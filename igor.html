What It Is

I.G.O.R. is a personal robotic operator — a self-hosted AI system built on Anthropic's Claude API and running on Oracle Cloud infrastructure. It operates continuously, handling research, scheduling, monitoring, memory management, and development tasks through a simple conversational interface.

Unlike a standard chatbot, I.G.O.R. doesn't wait for a human to confirm each step. It plans, acts, evaluates results, and iterates toward a goal on its own.

Architecture

The system runs on a ReAct (Reason-Act-Observe) loop with extended thinking enabled. At each iteration, I.G.O.R. reasons about the current state, selects and executes a tool, observes the result, and decides the next move — repeating until the goal is reached.

Extended thinking gives I.G.O.R. the ability to work through ambiguous, multi-constraint problems before committing to an action — producing more deliberate decisions rather than reactive ones.

Capabilities

  • Web search and live URL fetching for real-time information gathering
  • Python execution for computation, data processing, and scripting
  • Shell access for system-level operations and process control
  • File read/write for persistent output and document generation
  • Persistent memory across sessions for long-running context
  • Deep research mode — runs indefinitely on a question, accumulating findings and identifying unexplored threads
  • Self-modification — reads its own source, writes improvements, validates changes, and signals for redeployment

Deep Research Mode

Based on the autoresearch pattern, I.G.O.R.'s deep research mode runs without a fixed endpoint. Given a question or topic, it accumulates findings across iterations, identifies gaps in its own knowledge, pursues unexplored threads, and builds a comprehensive report — all without human guidance between cycles.

Self-Modification

I.G.O.R. can read its own source code, propose and write improvements, validate the changes, and signal for redeployment. This is a foundational step toward a fully autonomous self-improving operator — a system that gets better at its own job over time.
